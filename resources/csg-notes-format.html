CSG Notes Formatting

How to write compliant SEM_Notes__c entries

Template

MM/DD/YYYY Full Name #LicHashtag #SigHashtag [#RA shortened-URL] [#AFHashtag]
Brief narrative: current status, next steps, key milestones.

All fields on one continuous entry. No line breaks before the narrative. Date must be current month.

Field-by-Field Breakdown

PositionContentRequired?Notes
1Date (MM/DD/YYYY)YesMust be within the current reporting period
2CSM Full NameYesFirst and last name of the assigned CSM
3License hashtagYes#LicNoAttrit, #LicPartialAttrit, or #LicFullAttrit
4Signature hashtagYes (Sig accts)#NoSigRisk, #SigRisk, or #SigAttrit
5#RA + URLIf RA exists#RA followed by shortened Org62 link to the Red Account
6Agentforce hashtagIf applicable#AfNoRisk or #AFAttrit
7NarrativeYesConcise status, risks, and next actions

Good Examples

06/02/2026 Benjamin Batista #LicNoAttrit #NoSigRisk Go-live blocked by Cimulate dependency; R&D (Ata) prioritizing before July. KEM Memorial Day completed. MuleSoft SIP $35K pending.
06/02/2026 Benjamin Batista #LicNoAttrit #SigRisk #RA sfdc.co/abc123 CSM transition complete 5/12. AgentForce paused; restart in 30-45d. CTO requested June 24 mtg re: roadmap.
5/29/26 K.Martinez #LicNoAttrit #NoSigRisk #AfNoRisk New acct, onbd Mar'26. SAP go-live Jun3 (KEM active/ProM live). ATHR+UX queued post-SAP. On-site QBR done.

Bad Examples (Common Failures)

#NoAttrit #NoSigAttrit Customer renewed. Everything fine.
Problems: Missing date, missing CSM name, invalid hashtags (#NoAttrit, #NoSigAttrit), vague narrative.
RP(5/18): #NoAttrit. This was created to keep track of the renewal in Feb 2027.
Problems: Abbreviated name format, #NoAttrit is invalid (use #LicNoAttrit), missing Sig hashtag.
#SuccessDeskPremierUpsellQ4FY17
Problems: Stale entry from 2017. No current date, no CSM, no valid hashtags, no narrative.

Step-by-Step: Writing Your CSG Notes

1
Start with today's date in MM/DD/YYYY format
2
Add your full name (first + last)
3
Add one license hashtag: #LicNoAttrit, #LicPartialAttrit, or #LicFullAttrit
4
If Signature account, add: #NoSigRisk, #SigRisk, or #SigAttrit
5
If an active Red Account exists, add #RA followed by the shortened URL
6
Write 1-2 sentences: current status, key risks, and next actions
